Step 5. Design.

Design the poster.

Using an image found online I will paste it on my canvas by pressing CTRL + V

I will trim the black areas of the image using the Magic Wand in order to make them transparent

I made it transparent so that I can use the flower image as the background to create an effect

To fill the rest of the transparent areas in the poster I will use the rectangle tool to fill the background

I will merge all three layers together to create a single rasterized layer

Now I will paste the flower image and put it behind my background rasterized layer to complete the effect

Now I will add my text by using the Text Tool, the first text will be my Header which is "Jazz@Bar96"

I will make the "Jazz" text larger than the rest to create an effect

The Font used for this text is "Mermaid"

I will now create the details of this event using the text tool, the font i will use is "Kayak Sans"

I will then change the font size to 18px

To create the "JAZZ" texts on the side I will make the text the same color as the background image

I will then go to FX > Stroke > 2px to create the text effect

I will then add another color to my background and change the text to match the background color

Last I will add the bottom background color by using the Rectangle Tool
